# /complexity

Analyze code complexity metrics for the specified file or function:

## Analysis Includes

- **Cyclomatic Complexity**: Count of decision points
- **Cognitive Complexity**: Readability difficulty score
- **Function Length**: Lines of code per function
- **Nesting Depth**: Maximum indentation levels
- **Parameter Count**: Number of function parameters

## Arguments

- `$ARGUMENTS`: File path or function name to analyze

## Recommendations

Suggest refactoring strategies for high-complexity areas, including:

- Extract method opportunities
- Guard clause patterns
- Early returns
- Decomposition strategies
