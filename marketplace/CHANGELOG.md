# Changelog

All notable changes to this marketplace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-28

### Added

- Initial marketplace release with 6 comprehensive plugins
- **code-review-tools**: Commands, agents, skills, hooks, and output styles for code review
- **deployment-automation**: MCP server integration for deployment workflows
- **testing-suite**: TDD skills and test automation hooks
- **documentation-generator**: AI-powered documentation with docstring and README skills
- **security-scanner**: LSP server integration for security scanning
- **git-workflow**: Subagent hooks and git operation commands
- Complete documentation for all configuration fields
- Sample implementations for all plugin component types
